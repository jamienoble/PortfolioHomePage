How to Use

1. Add the canvas element:

<canvas id="starfield"></canvas>

2. Include the script:

<script src="starfield-background.js"></script>

3. (Optional) Customize:

// Custom configuration
StarfieldBackground.init('starfield', {
    starCount: 200,
    shootingStarChance: 0.005,
    showAries: true,
    ariesInterval: 300
});
